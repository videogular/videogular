vgScrubBarCuePoints
directive in module com.2fdevs.videogular.plugins.controls

Description

Layer inside vg-scrub-bar to display a cue point timeline.

<videogular vg-theme="config.theme.url">
   <vg-media vg-src="sources"></vg-media>

   <vg-controls>
       <vg-scrub-bar>
           <vg-scrub-bar-cue-points vg-cue-points='config.cuePoints[0]'></vg-scrub-bar-cue-points>
       </vg-scrub-bar>
   </vg-controls>
</videogular>

Usage

as element:
<vg-scrub-bar-cue-points>
</vg-scrub-bar-cue-points>
