vgControls
directive in module com.2fdevs.videogular.plugins.controls

Description

This directive acts as a container and you will need other directives to control the media. Inside this directive you can add other directives like vg-play-pause-button and vg-scrub-bar.

Usage

as element:
<vg-controls
       vg-autohide="{boolean=false}"
       vg-autohide-time="{number=2000}">
</vg-controls>

Parameters

ParamTypeDetails
vgAutohideboolean=false

Boolean variable or value to activate autohide.

vgAutohideTimenumber=2000

Number variable or value that represents the time in milliseconds that will wait vgControls until it hides.

<videogular vg-theme="config.theme.url">
   <vg-media vg-src="sources"></vg-media>

   <vg-controls vg-autohide='config.autohide' vg-autohide-time='config.autohideTime'></vg-controls>
</videogular>
