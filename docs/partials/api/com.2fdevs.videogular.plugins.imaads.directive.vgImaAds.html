vgImaAds
directive in module com.2fdevs.videogular.plugins.imaads

Description

Directive to show Google Interactive Media Ads.

Usage

as element:
<vg-ima-ads
       vg-network="{string}"
       vg-unit-path="{string}"
       vg-companion="{string}"
       vg-companion-size="{array}"
       vg-ad-tag-url="{string}"
       vg-skip-button="{string}">
</vg-ima-ads>

Parameters

ParamTypeDetails
vgNetworkstring

Your network name.

vgUnitPathstring

Target unit path.

vgCompanionstring

Companion name.

vgCompanionSizearray

Companion size as an array like [width, height].

vgAdTagUrlstring

Ad tag url, usually a link to the VAST file provided by DoubleClick.

vgSkipButtonstring

Custom skip button HTML like <div class='skipButton'>skip ad</div>.
