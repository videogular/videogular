vgCrossorigin
directive in module com.2fdevs.videogular

Description

Optional directive for vg-media to add or remove a crossorigin policy to the video object. Possible values are: "anonymous" and "use-credentials". This feature should be enabled if you want to have your subtitles or video files on a different domain than the video player. Additionally you need to add CORS policies to your video and track files to your server to make it work.

Usage

as attribute
<ANY vg-crossorigin>
   ...
</ANY>
