vgDash
directive in module com.2fdevs.videogular.plugins.dash

Description

Adds DASH support for vg-media. This plugin requires dash.all.js file available at dash.js project: https://github.com/Dash-Industry-Forum/dash.js

<videogular vg-theme="config.theme.url" vg-autoplay="config.autoPlay">
   <vg-media vg-src="sources" vg-dash></vg-media>
</videogular>

Usage

as attribute
<ANY vg-dash>
   ...
</ANY>
