vgVideo
service in module com.2fdevs.videogular

Description

Directive to add a source of videos. This directive will create a <video> tag and usually will be above plugin tags.

You can customize vgVideo with these attributes:

Usage

as element:
<vg-video
       vg-src="{array}"
       vg-loop="{boolean}"
       vg-preload="{string}"
       vg-native-controls="{boolean}"
       vg-tracks="{array}">
</vg-video>

Parameters

ParamTypeDetails
vgSrcarray

Bindable array with a list of video sources. A video source is an object with two properties src and type. The src property must contains a trusful url resource. {src: $sce.trustAsResourceUrl("http://www.videogular.com/assets/videos/videogular.mp4"), type: "video/mp4"} This parameter is required.

vgLoopboolean

vgLoop Boolean value or scope variable name to auto start playing video when it is initialized.

(default: false)

vgPreloadstring

vgPreload String value or scope variable name to set how to preload the video. This parameter is disabled in mobile devices because user must click on content to start data preload.

(default: false)

vgNativeControlsboolean

vgNativeControls String value or scope variable name to set native controls visible.

(default: false)

vgTracksarray

vgTracks Bindable array with a list of subtitles sources. A track source is an object with five properties: src, kind, srclang, label and default. {src: "assets/subs/pale-blue-dot.vtt", kind: "subtitles", srclang: "en", label: "English", default: "true/false"}

(default: false)
