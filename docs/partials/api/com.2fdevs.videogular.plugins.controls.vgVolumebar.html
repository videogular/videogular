vgVolumebar
directive in module com.2fdevs.videogular.plugins

Description

Directive to display a vertical volume bar to control the volume. This directive must be inside vg-volume directive and requires vg-mutebutton to be displayed.

<videogular vg-theme="config.theme.url">
   <vg-video vg-src="sources"></vg-video>

   <vg-controls vg-autohide='config.autohide' vg-autohide-time='config.autohideTime'>
       <vg-volume>
           <vg-mutebutton><vg-mutebutton>
           <vg-volumebar><vg-volumebar>
       </vg-volume>
   </vg-controls>
</videogular>

Usage

as element:
<vg-volumebar>
</vg-volumebar>
