vgMedia
directive in module com.2fdevs.videogular

Description

Directive to add a source of videos or audios. This directive will create a <video> or <audio> tag and usually will be above plugin tags.

Usage

as element:
<vg-media
       vg-src="{array}"
       vg-type="{string}">
</vg-media>

Parameters

ParamTypeDetails
vgSrcarray

Bindable array with a list of media sources. A media source is an object with two properties src and type. The src property must contains a trustful url resource.

vgTypestring

String with "video" or "audio" values to set a 

{
   src: $sce.trustAsResourceUrl("path/to/video/videogular.mp4"),
   type: "video/mp4"
}
