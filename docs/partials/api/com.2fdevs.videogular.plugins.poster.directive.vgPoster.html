vgPoster
directive in module com.2fdevs.videogular.plugins.poster

Description

Shows an image when player hasn't been played or has been completed a video.

Usage

as element:
<vg-poster
       vg-url="{string}">
</vg-poster>

Parameters

ParamTypeDetails
vgUrlstring

String with a scope name variable. URL to an image supported by the img tag. This parameter is required.

<videogular vg-theme="config.theme.url" vg-autoplay="config.autoPlay">
   <vg-video vg-src="sources"></vg-video>

   <vg-poster vg-url='config.plugins.poster.url'></vg-poster>
</videogular>
