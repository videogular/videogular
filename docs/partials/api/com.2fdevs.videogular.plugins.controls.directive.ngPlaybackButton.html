ngPlaybackButton
directive in module com.2fdevs.videogular.plugins.controls

Description

Directive to display a playback buttom to control the playback rate.

Usage

as element:
<ng-playback-button
       vg-speeds="{array}">
</ng-playback-button>

Parameters

ParamTypeDetails
vgSpeedsarray

Bindable array with a list of speed options as strings. Default ['0.5', '1', '1.5', '2'] 

<videogular vg-theme="config.theme.url">
   <vg-media vg-src="sources"></vg-media>

   <vg-controls vg-autohide='config.autohide' vg-autohide-time='config.autohideTime'>
       <vg-playback-button vg-speeds='config.playbackSpeeds'></vg-playback-button>
   </vg-controls>
</videogular>
