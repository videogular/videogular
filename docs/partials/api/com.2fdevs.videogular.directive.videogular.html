videogular
directive in module com.2fdevs.videogular

Description

Main directive that must wrap a <vg-media> tag and all plugins.

<video> tag usually will be above plugin tags, that's because plugins should be in a layer over the <video>.

Usage

as element:
<videogular
       vg-theme="{string}"
       vg-plays-inline="{boolean}"
       vg-clear-media-on-navigate="{boolean}"
       vg-auto-play="{boolean}"
       vg-start-time="{boolean}"
       vg-virtual-clip-duration="{boolean}"
       vg-cue-points="{object}"
       vg-config="{function}"
       vg-can-play="{function}"
       vg-complete="{function}"
       vg-update-volume="{function}"
       vg-update-playback="{function}"
       vg-update-time="{function}"
       vg-update-state="{function}"
       vg-player-ready="{function}"
       vg-change-source="{function}"
       vg-plays-inline="{function}"
       vg-native-fullscreen="{function}"
       vg-seeking="{function}"
       vg-seeked="{function}"
       vg-error="{function}">
</videogular>

Parameters

ParamTypeDetails
vgThemestring

String with a scope name variable. This directive will inject a CSS link in the header of your page. This parameter is required.

vgPlaysInlineboolean

vgPlaysInline Boolean value or a String with a scope name variable to use native fullscreen (default) or set fullscreen inside browser (true).

(default: false)

vgClearMediaOnNavigateboolean

vgClearMediaOnNavigate Boolean value or a String with a scope name variable to reset the video player when user navigates.

This is useful to allow continuous playback between different routes.

(default: true)

vgAutoPlayboolean

vgAutoPlay Boolean value or a String with a scope name variable to auto start playing video when it is initialized.

This parameter is disabled in mobile devices because user must click on content to prevent consuming mobile data plans.

(default: false)

vgStartTimeboolean

vgStartTime Number value or a String with a scope name variable to start playing the video at a certain time.

(default: -1)

vgVirtualClipDurationboolean

vgVirtualClipDuration Number value or a String with a scope name variable for a length to limit the video playback to.

(default: -1)

vgCuePointsobject

Bindable object containing a list of timelines with cue points objects. A timeline is an array of objects with the following properties:

  • timeLapse is an object with two properties start and end representing in seconds the period for this cue points.
  • onEnter callback called when user enters on a cue point. callback(currentTime, timeLapse, params)
  • onLeave callback called when user seeks backwards and leave the current cue point or a completed cue point. callback(currentTime, timeLapse, params)
  • onUpdate callback called when the current time is between timeLapse.start and timeLapse.end. callback(currentTime, timeLapse, params)
  • onComplete callback called when the user seek forward or the current time passes timeLapse.end property. callback(currentTime, timeLapse, params)
  • params an object with values available to receive in the callback..

vgCanPlayfunction

Function name in controller's scope to call when video is able to begin playback

vgCompletefunction

Function name in controller's scope to call when video have been completed.

vgUpdateVolumefunction

Function name in controller's scope to call when volume changes. Receives a param with the new volume.

vgUpdatePlaybackfunction

Function name in controller's scope to call when playback changes. Receives a param with the new playback rate.

vgUpdateTimefunction

Function name in controller's scope to call when video playback time is updated. Receives two params with current time and duration in milliseconds.

vgUpdateStatefunction

Function name in controller's scope to call when video state changes. Receives a param with the new state. Possible values are "play", "stop" or "pause".

vgPlayerReadyfunction

Function name in controller's scope to call when video have been initialized. Receives a param with the videogular API.

vgChangeSourcefunction

Function name in controller's scope to change current video source. Receives a param with the new video.

vgPlaysInlinefunction

Boolean to play video inline. Generally used in mobile devices.

vgNativeFullscreenfunction

Boolean to disable native fullscreen.

vgSeekingfunction

Function name in controller's scope to call when the video has finished jumping to a new time. Receives a param with the seeked time and duration in seconds.

vgSeekedfunction

Function name in controller's scope to call when the video is jumping to a new time. Receives two params with the seeked time and duration in seconds.

vgErrorfunction

Function name in controller's scope to receive an error from video object. Receives a param with the error event. This is a free parameter and it could be values like "new.mp4", "320" or "sd". This will allow you to use this to change a video or video quality. This callback will not change the video, you should do that by updating your sources scope variable.
